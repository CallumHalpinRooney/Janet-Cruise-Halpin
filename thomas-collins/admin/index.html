Connect the bookings database

The dashboard reads the bookings table in Supabase. If it's empty or missing, create it once with:

create table bookings ( id bigint generated always as identity primary key, created_at timestamptz default now(), service text, service_id text, price text, duration int, date date, date_label text, "time" text, name text, phone text, email text, notes text, source text ); -- allow the site to insert and this dashboard to read: alter table bookings enable row level security; create policy "insert" on bookings for insert to anon with check (true); create policy "read" on bookings for select to anon using (true);
